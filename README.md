# Carrinho de Compras

Sistema desenvolvido em JavaScript para simular um carrinho de compras com cadastro de produtos, aplicação de cupons e finalização de pedidos.

## Estrutura do projeto

- `dados.js` → Banco de dados simulado em memória
- `validacao.js` → Validações de produtos, quantidades e cupons
- `utils.js` → Funções utilitárias de cálculo e formatação
- `produtos.js` → Cadastro e listagem de produtos
- `carrinho.js` → Adição, remoção e visualização do carrinho
- `pedido.js` → Finalização e histórico de pedidos
- `index.js` → Demonstração completa do sistema

## Como executar
```bash
node index.js
```

## Funcionalidades

- Cadastrar produtos com nome, preço e estoque
- Adicionar e remover itens do carrinho
- Aplicar cupons de desconto
- Finalizar pedidos com atualização de estoque
- Visualizar histórico de pedidos
- Validação completa de dados
